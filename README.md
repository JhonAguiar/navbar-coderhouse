# App Coder House Tienda Virtual

## Descripción

El siguiente repositorio contiene la aplicacion desarrollada para Coderhouse durante el transcurso del curso de React.js

Durante el desarrollo de este curso se realizo el siguiente ecommerce en el cual se aplicaron los temas aprendidos durante la clase,

por medio de el proyecto se puede acceder a un carrito de compra en el cual yo puedo filtrar productos por categorias al comprar ver el detalle de ese producto con una descripción detallada.

Al tener mi carrito de compra puedo enviar las ordenes generadas a Firebase para almacenarlas y procesar esta información

![GIF de muestra.](/public/tienda.gif "Overview del ecommerce")

## Tecnoglogías utilizadas

Para el desarrollo del proyecto se utilizaron las siguientes tecnologias:

* [React JS](https://reactjs.org/)

* [Bootstrap](https://getbootstrap.com/)
* [Firebase](https://firebase.google.com/)
* [React](https://react-icons.github.io/react-icons/)


## Ejecución del proyecto

Para ejecutar el proyecto, el mismo puede descargarse como .zip o clonarlo con:

```
    git clone https://github.com/JhonAguiar/navbar-coderhouse.git

    cd navbar-coderhouse
```

Instalar las dependencias:

```
    npm install
```
